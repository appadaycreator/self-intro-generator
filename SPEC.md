# 自己紹介文ジェネレーター【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Self Intro Generator
**バージョン**: 1.1.0
**更新日**: 2026-05-29
**URL**: https://appadaycreator.com/self-intro-generator/

経歴・強み・目標を入力するだけで就活・転職・SNS用の自己紹介文を自動生成。登録不要・完全無料でご利用いただけます。

## 変更内容（v1.1.0）

- **P1: PWA theme-color統一** — `#7c3aed` → `#6366F1`（meta/manifest/UI全統一）
- **P1: コピートースト** — `alert()` を非侵襲的トースト通知に変更
- **P1: フォームバリデーション改善** — `alert()` をインラインエラー表示に変更
- **P1: twitter:title追加** — Twitter Card metaタグ補完
- **P2: 目標フィールド追加** — 「目標・アピールしたいこと」入力欄、各テンプレートに組み込み
- **P2: 文字数カウンター** — 用途別上限参考値付き文字数表示
- **P2: localStorage永続化** — ページリロード後も入力値を復元
- **P3: 生成文編集機能** — textareaで直接編集可能（auto-resize付き）

## データ管理

- **ストレージ**: ブラウザ localStorage（キー: `self-intro-inputs`）
- **保存タイミング**: 「自己紹介文を生成する」ボタン押下時
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## フォームフィールド

| フィールドID | 名称 | 必須 | 用途 |
|------------|------|------|------|
| name | 氏名 | ✅ | 全テンプレートに使用 |
| job | 職種・業種 | ✅ | 全テンプレートに使用 |
| years | 経歴年数 | - | 経験年数の文言に使用 |
| strength | 強み・得意なこと | - | 専門性の文言に使用 |
| hobby | 趣味・関心 | - | 人柄表現に使用 |
| goal | 目標・アピール | - | 将来志向の文言に使用 |
| usage | 用途 | ✅ | テンプレート種別の選択 |

## テンプレート種別

| 用途 | キー | 文字数上限目安 |
|-----|------|------------|
| 就活・転職 | job | 400字 |
| SNSプロフィール | sns | 160字（X上限） |
| 社内自己紹介 | internal | 300字 |
| 名刺・ビジネス交流 | card | 200字 |

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json）

- localStorage によるデータ永続化

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する
